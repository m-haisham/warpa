# rpalib

rpalib is a library that implements renpy archive standard for reading and writing. The project aims to provide an easy-to-use abstraction over renpy archives.

## Support

| Version | Read | Write |
| ------- | ---- | ----- |
| 3.2     | Yes  | No    |
| 3.0     | Yes  | Yes   |
| 2.0     | Yes  | Yes   |
| 1.0     | No   | No    |
